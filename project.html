1: PUBG Game Winner Prediction

Project: Game Winner Prediction (PUBG Dataset)
Type: Regression | Tools: Python, Pandas, CatBoost, Feature Engineering

Problem Statement:

In competitive online gaming, predicting a player’s likelihood of winning can provide valuable feedback to both developers and gamers. Using a dataset of 4.4 million records from PUBG matches, I built a predictive model to estimate each player’s final winning percentage (winPlacePerc).

Approach:

  • Preprocessed a massive dataset containing 4.4M+ rows.
  • Engineered new features like walk distance, kill count, headshots, survival rank, and match type grouping.
  • Applied CatBoost Regressor, known for handling categorical features well.
  • Tuned hyperparameters and removed data leaks and inconsistencies.

Results:

Achieved an R² score of 0.93 and RMSE = 0.08. Delivered a highly accurate model capable of ranking players effectively during or after a match.

Key Learnings:

  • Tackled high-volume data under memory constraints.
  • Improved feature engineering for gaming datasets.
  • Gained experience in tuning tree-based models (CatBoost).

2: Credit Score Prediction

Project: Credit Score Classification
Type: Classification | Tools: Gradient Boosting, SMOTE, Scikit-learn

Problem Statement:

Financial institutions must assess customer creditworthiness before lending. I created a classification model to categorize users into "Poor", "Standard", or "Good" credit score classes using behavioral and financial features.

Approach:

  • Cleaned and preprocessed a real-world credit dataset.
  • Detected and handled class imbalance using SMOTE (Synthetic Minority Oversampling).
  • Selected relevant features like loan history, income level, credit inquiries, and payment delays.
  • Trained and tuned a Gradient Boosting Classifier.

Results:

Achieved a Gini Coefficient of 1.0, indicating near-perfect model performance.

Key Learnings:

  • Improved data balance using advanced oversampling.
  • Deepened understanding of credit modeling in real-world applications.
  • Enhanced skills in model evaluation and feature impact analysis.
